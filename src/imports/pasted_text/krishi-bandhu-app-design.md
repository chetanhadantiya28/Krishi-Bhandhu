If you're using an AI UI builder like Figma AI, Lovable, Bolt, v0, Claude, Gemini, or Cursor, don't ask it to generate individual screens. Instead, give it a **complete product design brief** so it understands the entire application and maintains consistency across all screens.

Use the following prompt:

---

# Krishi Bandhu — Complete Mobile UI/UX Design Prompt

Create a **professional, production-ready Android mobile application** named **Krishi Bandhu**, an AI-powered agriculture assistant designed specifically for **small-scale Indian farmers**, **Agricultural Officers**, and **Government Agriculture Departments**.

The design must feel like a combination of Google Material 3, WhatsApp simplicity, and modern agriculture technology.

The interface should be extremely simple because many users have limited digital literacy.

The application should support **Hindi, English, Marathi, Gujarati, Punjabi, Bengali, Tamil, Telugu, Kannada, Malayalam, Odia, Assamese**, and other Indic languages.

---
